== Roof

Built using Ruby on Rails, uses Backbone.js views which are appended in different elements of a classic rails view. 
Uses Mapbox.js and Geocoder to localize a room, makes a search verifying location and dates availability, The User 
can make a request on the fly using jQuery UJS, When an user accepts a request, automatically reserve the dates of 
the room and eliminate any overlapping request, own design and style made by "hand" using only Css.
